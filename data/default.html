Landing pages for your SaaS with ease.

Destack is a page builder that runs locally and saves everything in the repository. It comes packed with different themes and tons of components and supports multiple pages, image uploading and forms.

Stop worrying about your landing pages and focus on the core of your product.

Features

Destack is based on Craft.js and comes with many tailwind components from other open source projects like Tailblocks and Hyper UI.

Components

Includes tons of well designed and heavily functional blocks from Tailblocks, Meraki UI and Hyper UI.

Custom Editor

Powered by a custom built editor based on Craft.js that favours simplicity and getting work done.

Own the code

Stores everything on the repo, including the generated pages and the uploaded assets. Has no external dependencies.

Assets upload

Stores all uploaded images in the repo. Also supports HTML and API forms out of the box.

Easy to setup

Works existing & new React and Next.js projects. You just need to create a new page or route and import the library.

Easy to deploy

No changes to deploy to production. Works with Vercel, Netlify and other Jamstack sites.
